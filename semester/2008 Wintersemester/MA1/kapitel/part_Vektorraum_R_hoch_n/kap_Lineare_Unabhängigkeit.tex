\chapter{Lineare Unabhängigkeit, Unterräume}\section{Definitionen}\subsection{Lineare Unabhängigkeit}Gegeben seien $k (k \in \mathbb{N})$ Vektoren $\vec{a_1}, \vec{a_2}, \vec{a_k} \in \mathbb{R}^n$. Diese Vektoren heißen linear unabhängig\index{Linear Unabhängig}, wenn für $\lambda_1, \lambda_2, \dots, \lambda_x \in \mathbb{R}$. Aus der Gleichung\[\lambda_1 \vec{a_1} + \lambda_2 \vec{a_2} + \lambda_3 \vec{a_3} + \dots + \lambda_k \vec{a_k} = \sumx{l = 1}{k}{\lambda_l \vec{a_l} = \vec{0}}\]folgt $\lambda_1 = \lambda2 = \dots = \lambda_k = 0$. Andernfalls nennt man $\vec{a_1}, \vec{a_2}, \dots, \vec{a_k}$ linear abhängig\index{Linear Abhängig}.\\Eine Menge $M \subseteq \mathbb{R}^n$ heißt linear unabhängig\index{Linear Unabhängig}, falls sie leer ist oder falls sie aus (endlicher vielen) linear unabhängigen Vektoren besteht. Andernfalls nennt man $M$ linear abhängig\index{Linear Abhängig}.\\\textalignenum{Bemerkung:}{}{\item Ein einzelner Vektor $\vec{v} \neq \vec{0}$ ist immer linear unabhängig.\item Jeder Menge $M \subseteq \mathbb{R}$ mit $\vec{0} \in M$ ist linear abhängig.}\subsection{Linearkombination}Sei $k \in \mathbb{N}, \vec{a_1}, \vec{a_2}, \dots, \vec{a_k} \in \mathbb{R}^n$ und $\lambda_1, \lambda_2, \dots, \lambda_k \in \mathbb{R}$ dann nennt man den Vektor\[\vec{v} = \lambda_1 \mal \vec{a_1} + \lambda_2 \mal \vec{a_2} + \dots + \lambda_k \mal \vec{a_k} = \sumx{l = 1}{k}{\lambda_l \mal \vec{a_l}}\]Linearkombination\index{Linearkombination} der $\vec{a_l}$ mit Gewichten $\lambda_l (1 \leq l \leq k)$\\\textalign{Bemerkung:}{Sind $\vec{a_1}, \vec{a_2}, \dots, \vec{a_k}$ linear abhängig, so gibt es eine Linearkombination die $\vec{0}$ ergibt und dabei mindestens ein $\lambda_l \neq 0$ enthält.\\Wenn $\lambda_1 \neq 0$ so folgt:\[\vec{a_1} = \frac{1}{\lambda_1} \mal \left(- \sumx{l = 2}{k}{\lambda_l \mal \vec{a_l}\right)} = - \frac{\lambda_2}{\lambda_1} \vec{a_2} - \frac{\lambda_3}{\lambda_1} \vec{a_3} - \frac{\lambda_n}{\lambda_1} \vec{a_k} = \sumx{l = 2}{k}{\left(-\frac{\lambda_l}{\lambda_k}\right) \mal \vec{a_l}}\]daher $\vec{a_1}$ ist eine Linearkombination der Vektoren $\vec{a_2}, \vec{a_3}, \dots, \vec{a_k}$\\Lässt sich umgekehrt einer der Vektoren $\vec{a_1}, \vec{a_2}, \dots, \vec{a_k}$ als Linearkombination der anderen Vektoren schreiben, dann sind die Vektoren $\vec{a_1}, \vec{a_2}, \dots, \vec{a_k}$ linear abhängig.}\subsection{Einheitsvektoren}Sei $\vec{e_i} (1 \leq i \leq n)$ der Vektor, dessen $i$-te Komplemente den Wert $1$ hat und alle anderen Komponenten $0$ sind.\[\vec{e_i} =(0, \dots, 0, \underbrace{1}_{\mbox{$r$-te Stelle}}, 0, \dots, 0)^T = \left(\begin{array}{c}0\\0\\1\\0\\0\end{array}\right)\leftarrow \mbox{$i$-te Stelle}\]Man nennt $\vec{e_i}$ den $i$-ten Einheitsvektor\index{Einheitsvektor}\\Für alle $\lambda_1, \lambda_2, \dots, \lambda_n \in \mathbb{R}$ gilt\[\lambda_1 \vec{e_1} + \lambda_2 \vec{e_2} + \dots + \lambda_n \vec{e_n} = \sumx{i = 1}{n}{\lambda_i \vec{e_i}} = \left(\begin{array}{c}\lambda_1\\\lambda_2\\\vdots\\\lambda_n\end{array}\right)\]Folglich sind $\vec{e_1}, \vec{e_2}, \dots, \vec{e_n}$ linear unabhängig\subsection{Unterraum}Eine nicht leere Menge $U \subseteq \mathbb{R}^n$ heißt Unterraum\index{Unterraum} von $\mathbb{R}^n$, falls gilt:\begin{enumerate}\item $\vec{a}, \vec{b} \in U \Ra \vec{a} + \vec{b} \in U$ (Abgeschlossenheit bezüglich Addition)\item $\vec{a} \in U, \lambda \in \mathbb{R} \Ra \lambda \mal \vec{a} \in U$ (Abgeschlossenheit bezüglich Multiplikation mit einem Skalar)\end{enumerate}Unterräume können aus einer Menge $M \subseteq \mathbb{R}^n$ konstruiert werden. Man definiert die Menge:\[U = \gklamm{\vec{u} \in \mathbb{R}^n \vert \vec{u}} = \sumx{i = 1}{k}{\lambda_1 \mal \vec{a_i}, \vec{a_i} \in M, \lambda_i \in \mathbb{R}, k \in \mathbb{N}}\]Man kann zeigen, dass $U$ ein Unterraum von $\mathbb{R}^n$ ist. Man schreibt\[U = \mbox{Span($M$)}\index{Span}\]und sagt, dass $M$ den Unterraum $U$ aufspannt, beziehungsweise dass $M$ ein Erzeugendesystem\index{Erzeugendesystem} von $U$ ist.\\\textalignenum{Bemerkung:}{}{\item $\vec{0}$ ist in jedem Unterraum enthalten\item $\gklamm{\vec{0}}$ ist der kleinste Unterraum von $\mathbb{R}^n$\\		$\mathbb{R}^n$ ist der größte Unterraum von $\mathbb{R}$}\subsection{Basis und Dimension}Sei $U$ ein Unterraum von $\mathbb{R}^n$\begin{enumerate}\item Die maximale Anzahl $m$ linear unabhängiger Vektoren in $U$ heißt Dimension\index{Dimension} von $U$. Man schreibt $m = \mbox{dim($U$)}$\item Hat $U$ die Dimension $m \in \gklamm{1, 2, \dots, m}$ und sind $\vec{a_1}, \vec{a_2}, \dots, \vec{a_m}$ linear unabhängige Vektoren aus $U$, so nennt man die Menge $\gklamm{\vec{a_1}, \vec{a_2}, \vec{a_3}, \dots, \vec{a_m}}$ Basis\index{Basis} von $U$\\Im Fall $U = \gklamm{\vec{0}}$ vereinbart man, dass $\emptyset$ die Basis von $U$ ist.\end{enumerate}\textalignenum{Bemerkung:}{}{\item Eine Basis ist ein minimales Erzeugendensystem.\item Wenn $\gklamm{\vec{a_1}, \dots, \vec{a_m}}$ eine Basis von $U$ ist, kann man jedes \vec{a_2}, \dots, \vec{a_{n + 1}}, \vec{a_i} \neq 0$ seien beliebig gewählt.\\(wäre $\vec{a_i} = 0$ für ein $i$, würde die Aussage sofort folgen)\\Wir betrachten die Gleichung\[\lambda_1 \vec{a_1} + \lambda_2 \vec{a_2} + \dots + \lambda_n \vec{a_{n + 1}} + \lambda_1 \vec{a_{n + 1}} = \vec{0}\]Da zwei Vektoren genau dann gleich sind, wenn alle Komponenten gleich sind, ergibt sich ein lineares Gleichungssystem mit $n$ Gleichungen in den Variablen $\lambda_1, \lambda_2, \dots, \lambda_n, \lambda_{n + 1}$\\Mit Hilfe des Gaussverfahrens kann man die erweiterte Koeffizientenmatrix auf die reduzierte Zeilenstufenform $U$ bringen. Da die rechte Seite von $U$ $\vec{0}$ ist, ist die letzte Spalte von $U$ sicher keine Pivot-Spalte und damit ist das zugehörige Gleichungssystem konsistent. Da wir mehr Variablen als Gleichungen haben, gibt es mindestens eine Freie Variable und damit hat sich das Gleichungssystem nach Satz \vref{sec:ExistenzundEindeutigkeitderLoesung} (2.) unendlich viele Lösungen insbesondere gibt es also eine Lösung, bei der $\lambda_k \neq 0$ für mindestens ein $k$ gilt\\\demonstrand{Damit sind $\vec{a_1}, \vec{a_2}, \dots, \vec{a_{n + 1}}$ linear abhängig.}}\subsection{Lösungsmengen von Vektorgleichungen}Die Lösungesmenge der Vektorgleichung $\vec{b} = \sumx{j = 1}{k}{x_j \mal \vec{a_j}}$ stimmt mit der Lösungsmenge des linearen
